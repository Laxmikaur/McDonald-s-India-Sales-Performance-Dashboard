# 🍔 McDonald's Sales Analysis Dashboard | Excel Project

## 📌 Project Overview
This project is an interactive **McDonald's Sales Analysis Dashboard** created in **Microsoft Excel**.  
The dashboard provides meaningful insights into sales, profit, customer orders, and product performance using charts, maps, slicers, and pivot tables.

The main goal of this project is to analyze business performance and present data in a visually interactive format for better decision-making.

---

## 🎯 Project Objectives
- Analyze total sales and profit performance
- Track month-wise sales trends
- Compare state-wise sales and profit
- Analyze category-wise quantity sold
- Understand gender-wise customer order distribution

---

### 🔹 Visualizations

---

## 📈 Key Insights
- Uttar Pradesh generated the highest profit.
- Monthly sales analysis helps identify peak sales periods.
- Burger and Fries categories contribute significantly to total orders.
- Sales distribution varies across different states.
- Customer order distribution is balanced across genders.
